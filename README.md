# Sales Forecasting System



## Overview

This project demonstrates how Artificial Intelligence can be used to forecast future sales using historical data. A Machine Learning model based on Linear Regression is implemented to identify patterns and predict upcoming sales values.

## Objective

To build a predictive system that helps businesses estimate future sales trends for better planning, inventory control, and decision-making.

## Problem Statement

Businesses often struggle to accurately predict future demand. Traditional methods often fail to capture trends effectively. This project uses a data-driven AI approach to provide reliable sales forecasting.

## Key Features

* Data preprocessing and transformation
* Time-based feature engineering
* Machine Learning model training
* Multi-day future sales prediction
* Graphical visualization of trends
* Excel export for business use

## Dataset Description

The dataset contains:

* date: Represents the day of sales
* daily_revenue: Total sales value

## Approach

1. Data creation and preprocessing
2. Conversion of date into numerical format
3. Feature engineering using day number
4. Model training using Linear Regression
5. Prediction of future sales values
6. Visualization of results
7. Exporting results to Excel

## Model Used

Linear Regression

## Results

The model successfully identifies the upward trend in sales data and predicts future values. The prediction results indicate consistent growth, which helps in planning future strategies.
